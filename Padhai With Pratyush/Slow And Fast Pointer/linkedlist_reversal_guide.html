1пёҸвғЈ Introduction - The Reversal Pattern

рҹ”§ The Core Concept

Linked List reversal = rewiring arrows (next pointers),
NOT swapping values!

рҹҺҜ The "Mechanic & Helper" Analogy

Think of rewiring train cars:

рҹ”§ Mechanic (curr): Moves forward
рҹӨқ Helper (prev): Holds the connection
рҹ“Ұ Storage (next): Saves where to go

Rule: Always save next BEFORE breaking connection!

/**
 * Reverse Nodes in K-Group
 * LeetCode 25
 * Time: O(N), Space: O(1)
 */

class Solution {
    // Helper: Reverse k nodes starting from head
    private void reverse(ListNode head, int k) {
        ListNode curr = head;
        ListNode prev = null;
        ListNode next = null;
        
        while (k > 0) {
            next = curr.next;      // Save next
            curr.next = prev;      // Reverse arrow в”Җв”ҖвҶ’ to вҶҗв”Җв”Җ
            prev = curr;           // Move prev forward
            curr = next;           // Move curr forward
            k--;
        }
    }
    
    public ListNode reverseKGroup(ListNode head, int k) {
        if (head == null) return null;
        
        ListNode left = head;
        ListNode right = null;
        ListNode prevLeft = null;
        ListNode result = null;
        
        while (true) {
            right = left;
            
            // Find right (k-1 steps from left)
            for (int i = 0; i < k - 1; i++) {
                if (right == null) break;
                right = right.next;
            }
            
            // Not enough nodes? Connect and break
            if (right == null) {
                if (prevLeft != null) {
                    prevLeft.next = left;
                }
                break;
            }
            
            ListNode nextLeft = right.next;  // Save!
            reverse(left, k);               // Reverse group
            
            // Connect previous group to this group
            if (prevLeft != null) {
                prevLeft.next = right;
            }
            
            if (result == null) result = right;
            
            prevLeft = left;
            left = nextLeft;
            
            if (left == null) break;
        }
        
        return result == null ? head : result;
    }
}

рҹ’Ў Key Takeaways:

  • вң“ Arrows reverse: в”Җв”ҖвҶ’ becomes вҶҗв”Җв”Җ within group
  • вң“ After reversal: right is new front, left is new back
  • вң“ prevLeft connects groups together
  • вң“ Always save nextLeft before reversing!

2. Make TWO critical connections:
   вҖў Connection 1: before (10) в”Җв”ҖвҶ’ new head (40)
   вҖў Connection 2: old left (20) в”Җв”ҖвҶ’ after (50)

public ListNode reverseBetween(ListNode head, int left, int right) {
    if (head == null || left == right) return head;
    
    ListNode t = head;
    ListNode before = null;
    int pos = 1;
    
    // Find "before" node
    while (t != null && pos < left) {
        before = t;
        t = t.next;
        pos++;
    }
    
    // Reverse segment
    ListNode curr = t;
    ListNode prev = null;
    ListNode next = null;
    int count = right - left + 1;
    
    while (count > 0) {
        next = curr.next;
        curr.next = prev;
        prev = curr;
        curr = next;
        count--;
    }
    
    // Reconnect
    t.next = curr;  // old left вҶ’ after
    
    if (before != null) {
        before.next = prev;  // before вҶ’ new head
        return head;
    } else {
        return prev;  // left==1 case
    }
}

public ListNode rotateRight(ListNode head, int k) {
    if (head == null) return null;
    
    // Find length and last node
    ListNode last = head;
    int n = 1;
    while (last.next != null) {
        last = last.next;
        n++;
    }
    
    // Optimize k
    k = k % n;
    if (k == 0) return head;
    
    // Find new tail at position (n-k)
    ListNode newTail = head;
    for (int i = 1; i < n - k; i++) {
        newTail = newTail.next;
    }
    
    // Perform rotation
    ListNode newHead = newTail.next;  // Save new head
    last.next = head;                 // Create ring
    newTail.next = null;              // Break ring
    
    return newHead;
}
