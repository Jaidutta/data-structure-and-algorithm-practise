1. Definition of a Graph with Visualization

Graph (G): A graph is a mathematical structure consisting of a set of vertices (also called nodes) and a set of edges that connect pairs of vertices.

Formal Definition: G = (V, E) where:

  • V = {A, B, C, D} - Set of vertices/nodes
  • E = {(A,B), (B,C), (C,D), (D,A)} - Set of edges connecting vertices
Visual Representation of G = (V, E)

V = {A, B, C, D} | E = {(A,B), (B,C), (C,D), (D,A), (A,C)}

2. Key Terminology with Visual Examples

Vertex (Node)

Each circle represents a vertex/node in the graph

Edge (Connection)

Lines connecting vertices are edges

Degree of a Vertex

Degree = number of edges connected to a vertex

Adjacent Vertices

Vertices connected by an edge are adjacent

Path

A sequence of vertices: A → B → C → D

Cycle

A path that starts and ends at the same vertex

Connected Graph

Path exists between every pair of vertices

Disconnected Graph

Separate components with no connecting paths

5. Java Code Examples

Basic Graph Class (Adjacency List)

Algorithm Explanation:

Data Structure: We use a HashMap where each key is a vertex (String) and each value is an ArrayList of adjacent vertices.

Adding a Vertex:

  1. Check if the vertex already exists in the adjacency list
  2. If not, create a new empty ArrayList for this vertex

Adding an Edge (Undirected):

  1. First ensure both vertices exist (add them if they don't)
  2. Add vertex2 to vertex1's adjacency list
  3. Add vertex1 to vertex2's adjacency list (since it's undirected)

Removing an Edge:

  1. Get the adjacency list for vertex1
  2. Remove vertex2 from it
  3. Get the adjacency list for vertex2
  4. Remove vertex1 from it

Removing a Vertex:

  1. Loop through all adjacency lists and remove this vertex from each
  2. Delete the vertex's own adjacency list from the map
import java.util.*;

/**
 * Graph class using Adjacency List representation
 * Each vertex stores a list of its adjacent vertices
 */
public class Graph {
    // Map: vertex name -> list of adjacent vertices
    private Map<String, List<String>> adjacencyList;
    
    /**
     * Constructor - Initialize empty adjacency list
     */
    public Graph() {
        this.adjacencyList = new HashMap<>();
    }
    
    /**
     * Add a vertex to the graph
     * Uses putIfAbsent to avoid overwriting existing vertex
     * @param vertex - the vertex to add
     */
    public void addVertex(String vertex) {
        adjacencyList.putIfAbsent(vertex, new ArrayList<>());
    }
    
    /**
     * Add an undirected edge between two vertices
     * The edge goes both ways: vertex1 <-> vertex2
     * @param vertex1 - first vertex
     * @param vertex2 - second vertex
     */
    public void addEdge(String vertex1, String vertex2) {
        // Ensure both vertices exist in the graph
        addVertex(vertex1);
        addVertex(vertex2);
        
        // Add edge in both directions (undirected)
        adjacencyList.get(vertex1).add(vertex2);
        adjacencyList.get(vertex2).add(vertex1);
    }
    
    /**
     * Remove an undirected edge between two vertices
     * @param vertex1 - first vertex
     * @param vertex2 - second vertex
     */
    public void removeEdge(String vertex1, String vertex2) {
        // Get adjacency lists for both vertices
        List<String> v1List = adjacencyList.get(vertex1);
        List<String> v2List = adjacencyList.get(vertex2);
        
        // Remove each vertex from the other's list
        if (v1List != null) v1List.remove(vertex2);
        if (v2List != null) v2List.remove(vertex1);
    }
    
    /**
     * Remove a vertex and all edges connected to it
     * @param vertex - the vertex to remove
     */
    public void removeVertex(String vertex) {
        // First, remove this vertex from all other vertices' adjacency lists
        adjacencyList.values().forEach(list -> list.remove(vertex));
        
        // Then remove the vertex's own entry from the map
        adjacencyList.remove(vertex);
    }
    
    /**
     * Display the graph in adjacency list format
     * Format: vertex -> neighbor1, neighbor2, ...
     */
    public void display() {
        for (String vertex : adjacencyList.keySet()) {
            System.out.print(vertex + " -> ");
            System.out.println(String.join(", ", adjacencyList.get(vertex)));
        }
    }
    
    /**
     * Get all vertices in the graph
     * @return Set of all vertex names
     */
    public Set<String> getVertices() {
        return adjacencyList.keySet();
    }
    
    /**
     * Get all neighbors (adjacent vertices) of a given vertex
     * @param vertex - the vertex whose neighbors we want
     * @return List of neighbor vertices
     */
    public List<String> getNeighbors(String vertex) {
        return adjacencyList.getOrDefault(vertex, new ArrayList<>());
    }
    
    /**
     * Check if an edge exists between two vertices
     * @param vertex1 - first vertex
     * @param vertex2 - second vertex
     * @return true if edge exists, false otherwise
     */
    public boolean hasEdge(String vertex1, String vertex2) {
        List<String> neighbors = adjacencyList.get(vertex1);
        return neighbors != null && neighbors.contains(vertex2);
    }
    
    /**
     * Get the degree of a vertex (number of edges connected to it)
     * @param vertex - the vertex whose degree we want
     * @return degree of the vertex
     */
    public int getDegree(String vertex) {
        List<String> neighbors = adjacencyList.get(vertex);
        return neighbors != null ? neighbors.size() : 0;
    }
    
    /**
     * Main method - demonstrates graph operations
     */
    public static void main(String[] args) {
        // Create a new graph
        Graph graph = new Graph();
        
        // Add vertices to the graph
        graph.addVertex("A");
        graph.addVertex("B");
        graph.addVertex("C");
        graph.addVertex("D");
        
        // Add edges between vertices
        graph.addEdge("A", "B");
        graph.addEdge("A", "C");
        graph.addEdge("B", "D");
        graph.addEdge("C", "D");
        
        // Display the graph structure
        graph.display();
        
        // Output:
        // A -> B, C
        // B -> A, D
        // C -> A, D
        // D -> B, C
        
        // Test various graph operations
        System.out.println("Degree of A: " + graph.getDegree("A")); // 2
        System.out.println("Neighbors of B: " + graph.getNeighbors("B")); // [A, D]
        System.out.println("Has edge A-C: " + graph.hasEdge("A", "C")); // true
    }
}

Directed Graph Class

Algorithm Explanation:

Directed Edge: Unlike undirected graphs, we only add the edge in one direction (source → destination).

Path Detection Algorithm (DFS-based):

  1. If current vertex equals destination, return true (path found)
  2. If current vertex is already visited, return false (avoid cycles)
  3. Mark current vertex as visited
  4. For each neighbor of current vertex:
    • Recursively check if there's a path from neighbor to destination
    • If any path is found, return true
  5. If no path found through any neighbor, return false

Cycle Detection Algorithm (DFS with Recursion Stack):

  1. Maintain two sets: visited (all seen vertices) and recursionStack (current path)
  2. For each unvisited vertex, start DFS
  3. In DFS:
    • Mark vertex as visited and add to recursion stack
    • For each neighbor:
      • If neighbor not visited: recursively visit it
      • If neighbor is in recursion stack: cycle detected! Return true
    • Remove vertex from recursion stack (backtrack)
  4. If no cycle found in any DFS, return false

In-Degree: Count how many edges point TO this vertex (count occurrences in all adjacency lists)

Out-Degree: Count how many edges point FROM this vertex (size of its adjacency list)

import java.util.*;

/**
 * Directed Graph class
 * Edges have direction: source -> destination
 */
public class DirectedGraph {
    // Adjacency list: vertex -> list of vertices it points to
    private Map<String, List<String>> adjacencyList;
    
    /**
     * Constructor - Initialize empty directed graph
     */
    public DirectedGraph() {
        this.adjacencyList = new HashMap<>();
    }
    
    /**
     * Add a vertex to the graph
     * @param vertex - the vertex to add
     */
    public void addVertex(String vertex) {
        adjacencyList.putIfAbsent(vertex, new ArrayList<>());
    }
    
    /**
     * Add a directed edge from source to destination
     * Edge only goes one way: source -> destination
     * @param source - starting vertex of the edge
     * @param destination - ending vertex of the edge
     */
    public void addEdge(String source, String destination) {
        // Ensure both vertices exist
        addVertex(source);
        addVertex(destination);
        
        // Add edge only in one direction
        adjacencyList.get(source).add(destination);
    }
    
    /**
     * Remove a directed edge
     * @param source - starting vertex
     * @param destination - ending vertex
     */
    public void removeEdge(String source, String destination) {
        List<String> neighbors = adjacencyList.get(source);
        if (neighbors != null) {
            neighbors.remove(destination);
        }
    }
    
    /**
     * Calculate in-degree of a vertex
     * In-degree = number of edges coming INTO this vertex
     * @param vertex - the vertex to check
     * @return in-degree count
     */
    public int getInDegree(String vertex) {
        int count = 0;
        // Count how many vertices have an edge to this vertex
        for (List<String> neighbors : adjacencyList.values()) {
            if (neighbors.contains(vertex)) {
                count++;
            }
        }
        return count;
    }
    
    /**
     * Calculate out-degree of a vertex
     * Out-degree = number of edges going OUT FROM this vertex
     * @param vertex - the vertex to check
     * @return out-degree count
     */
    public int getOutDegree(String vertex) {
        List<String> neighbors = adjacencyList.get(vertex);
        return neighbors != null ? neighbors.size() : 0;
    }
    
    /**
     * Check if there's a path from source to destination using DFS
     * @param source - starting vertex
     * @param destination - target vertex
     * @return true if path exists, false otherwise
     */
    public boolean hasPath(String source, String destination) {
        Set<String> visited = new HashSet<>();
        return dfsPath(source, destination, visited);
    }
    
    /**
     * Helper method: DFS to find path
     * @param current - current vertex in traversal
     * @param destination - target vertex
     * @param visited - set of already visited vertices
     * @return true if path found
     */
    private boolean dfsPath(String current, String destination, Set<String> visited) {
        // Base case: reached destination
        if (current.equals(destination)) return true;
        
        // Base case: already visited (avoid infinite loops)
        if (visited.contains(current)) return false;
        
        // Mark current as visited
        visited.add(current);
        
        // Recursively check all neighbors
        List<String> neighbors = adjacencyList.get(current);
        if (neighbors != null) {
            for (String neighbor : neighbors) {
                if (dfsPath(neighbor, destination, visited)) {
                    return true; // Path found through this neighbor
                }
            }
        }
        
        return false; // No path found
    }
    
    /**
     * Detect if the graph contains a cycle
     * Uses DFS with a recursion stack
     * @return true if cycle exists, false otherwise
     */
    public boolean hasCycle() {
        Set<String> visited = new HashSet<>();       // All visited vertices
        Set<String> recursionStack = new HashSet<>(); // Current path vertices
        
        // Check each vertex (handles disconnected components)
        for (String vertex : adjacencyList.keySet()) {
            if (!visited.contains(vertex)) {
                if (hasCycleDFS(vertex, visited, recursionStack)) {
                    return true; // Cycle found
                }
            }
        }
        
        return false; // No cycle in entire graph
    }
    
    /**
     * Helper method: DFS for cycle detection
     * @param vertex - current vertex
     * @param visited - all visited vertices
     * @param recursionStack - vertices in current DFS path
     * @return true if cycle detected
     */
    private boolean hasCycleDFS(String vertex, Set<String> visited, 
                                Set<String> recursionStack) {
        // Add to visited and current path
        visited.add(vertex);
        recursionStack.add(vertex);
        
        // Check all neighbors
        List<String> neighbors = adjacencyList.get(vertex);
        if (neighbors != null) {
            for (String neighbor : neighbors) {
                // If neighbor not visited, recurse
                if (!visited.contains(neighbor)) {
                    if (hasCycleDFS(neighbor, visited, recursionStack)) {
                        return true; // Cycle found in recursion
                    }
                } 
                // If neighbor is in current path, we have a cycle!
                else if (recursionStack.contains(neighbor)) {
                    return true; // Back edge detected = cycle
                }
            }
        }
        
        // Remove from recursion stack (backtrack)
        recursionStack.remove(vertex);
        return false;
    }
    
    /**
     * Display the directed graph
     */
    public void display() {
        for (String vertex : adjacencyList.keySet()) {
            System.out.print(vertex + " -> ");
            System.out.println(String.join(", ", adjacencyList.get(vertex)));
        }
    }
    
    /**
     * Main method - demonstrates directed graph operations
     */
    public static void main(String[] args) {
        DirectedGraph digraph = new DirectedGraph();
        
        // Create a directed cycle: A -> B -> C -> D -> A
        digraph.addEdge("A", "B");
        digraph.addEdge("B", "C");
        digraph.addEdge("C", "D");
        digraph.addEdge("D", "A");
        
        // Display the graph
        digraph.display();
        
        // Test various operations
        System.out.println("Has path from A to C: " + digraph.hasPath("A", "C")); // true
        System.out.println("Has cycle: " + digraph.hasCycle()); // true
        System.out.println("In-degree of C: " + digraph.getInDegree("C")); // 1 (from B)
        System.out.println("Out-degree of C: " + digraph.getOutDegree("C")); // 1 (to D)
    }
}

Weighted Graph Class

Algorithm Explanation:

Data Structure: Adjacency list where each entry contains Edge objects (destination vertex + weight).

Dijkstra's Shortest Path Algorithm:

This algorithm finds the shortest path from a start vertex to all other vertices in a weighted graph with non-negative weights.

  1. Initialize:
    • Set distance to start vertex = 0
    • Set distance to all other vertices = infinity
    • Create a priority queue (min-heap) ordered by distance
    • Add start vertex to priority queue
  2. Main Loop: While priority queue is not empty:
    • Extract vertex with minimum distance (current)
    • If already visited, skip it
    • Mark current as visited
    • For each neighbor of current:
      • Calculate: newDistance = distance[current] + edge_weight
      • If newDistance < distance[neighbor]:
        • Update distance[neighbor] = newDistance
        • Add neighbor to priority queue with new distance
  3. Result: The distances map contains shortest distances from start to all vertices

Why it works: By always processing the closest unvisited vertex, we guarantee that when we visit a vertex, we've found the shortest path to it.

Time Complexity: O((V + E) log V) with priority queue

import java.util.*;

/**
 * Edge class to represent a weighted edge
 * Stores destination vertex and weight
 */
class Edge {
    String destination;  // Where the edge goes to
    int weight;          // Cost/weight of the edge
    
    public Edge(String destination, int weight) {
        this.destination = destination;
        this.weight = weight;
    }
}

/**
 * Weighted Graph class
 * Each edge has an associated weight/cost
 */
public class WeightedGraph {
    // Adjacency list: vertex -> list of weighted edges
    private Map<String, List<Edge>> adjacencyList;
    
    /**
     * Constructor - Initialize empty weighted graph
     */
    public WeightedGraph() {
        this.adjacencyList = new HashMap<>();
    }
    
    /**
     * Add a vertex to the graph
     * @param vertex - the vertex to add
     */
    public void addVertex(String vertex) {
        adjacencyList.putIfAbsent(vertex, new ArrayList<>());
    }
    
    /**
     * Add a weighted undirected edge between two vertices
     * @param vertex1 - first vertex
     * @param vertex2 - second vertex
     * @param weight - weight/cost of the edge
     */
    public void addEdge(String vertex1, String vertex2, int weight) {
        // Ensure both vertices exist
        addVertex(vertex1);
        addVertex(vertex2);
        
        // Add edge in both directions with same weight (undirected)
        adjacencyList.get(vertex1).add(new Edge(vertex2, weight));
        adjacencyList.get(vertex2).add(new Edge(vertex1, weight));
    }
    
    /**
     * Display the weighted graph
     * Format: vertex -> neighbor1(weight1), neighbor2(weight2), ...
     */
    public void display() {
        for (String vertex : adjacencyList.keySet()) {
            System.out.print(vertex + " -> ");
            List<String> connections = new ArrayList<>();
            for (Edge edge : adjacencyList.get(vertex)) {
                connections.add(edge.destination + "(" + edge.weight + ")");
            }
            System.out.println(String.join(", ", connections));
        }
    }
    
    /**
     * Dijkstra's Algorithm - Find shortest paths from start vertex to all others
     * Only works with non-negative edge weights
     * @param start - starting vertex
     * @return Map of vertex -> shortest distance from start
     */
    public Map<String, Integer> dijkstra(String start) {
        // Store shortest distances from start to each vertex
        Map<String, Integer> distances = new HashMap<>();
        
        // Priority queue to process vertices by distance (min-heap)
        PriorityQueue<Node> pq = new PriorityQueue<>(
            Comparator.comparingInt(node -> node.distance)
        );
        
        // Track visited vertices
        Set<String> visited = new HashSet<>();
        
        // Initialize: distance to start = 0, all others = infinity
        for (String vertex : adjacencyList.keySet()) {
            distances.put(vertex, Integer.MAX_VALUE);
        }
        distances.put(start, 0);
        pq.offer(new Node(start, 0));
        
        // Main loop: process vertices in order of increasing distance
        while (!pq.isEmpty()) {
            Node current = pq.poll();
            String currentVertex = current.vertex;
            
            // Skip if already processed
            if (visited.contains(currentVertex)) continue;
            visited.add(currentVertex);
            
            // Check all neighbors
            List<Edge> neighbors = adjacencyList.get(currentVertex);
            if (neighbors != null) {
                for (Edge edge : neighbors) {
                    if (!visited.contains(edge.destination)) {
                        // Calculate new distance through current vertex
                        int newDist = distances.get(currentVertex) + edge.weight;
                        
                        // Update if we found a shorter path
                        if (newDist < distances.get(edge.destination)) {
                            distances.put(edge.destination, newDist);
                            pq.offer(new Node(edge.destination, newDist));
                        }
                    }
                }
            }
        }
        
        return distances;
    }
    
    /**
     * Helper class for Dijkstra's algorithm
     * Represents a vertex with its current known distance
     */
    static class Node {
        String vertex;
        int distance;
        
        public Node(String vertex, int distance) {
            this.vertex = vertex;
            this.distance = distance;
        }
    }
    
    /**
     * Main method - demonstrates weighted graph operations
     */
    public static void main(String[] args) {
        WeightedGraph graph = new WeightedGraph();
        
        // Build a weighted graph
        graph.addEdge("A", "B", 4);
        graph.addEdge("A", "C", 2);
        graph.addEdge("B", "E", 3);
        graph.addEdge("C", "D", 2);
        graph.addEdge("C", "F", 4);
        graph.addEdge("D", "E", 3);
        graph.addEdge("D", "F", 1);
        graph.addEdge("E", "F", 1);
        
        // Display the graph
        graph.display();
        
        // Find shortest paths from A to all other vertices
        System.out.println("\nShortest distances from A:");
        Map<String, Integer> distances = graph.dijkstra("A");
        for (String vertex : distances.keySet()) {
            System.out.println("A to " + vertex + ": " + distances.get(vertex));
        }
        // Example output: A to E: 7 (path: A->C->D->E)
    }
}

Graph Traversal: Depth-First Search (DFS)

Algorithm Explanation:

Depth-First Search (DFS): Explores as far as possible along each branch before backtracking.

DFS Recursive Algorithm:

  1. Start at the given vertex
  2. Mark current vertex as visited and add to result
  3. For each unvisited neighbor of current vertex:
    • Recursively call DFS on that neighbor
  4. When no more unvisited neighbors, backtrack to previous vertex

DFS Iterative Algorithm (using Stack):

  1. Create an empty stack and push start vertex
  2. Mark start vertex as visited
  3. While stack is not empty:
    • Pop vertex from stack and add to result
    • For each unvisited neighbor:
      • Mark neighbor as visited
      • Push neighbor onto stack

Key Idea: Stack (LIFO) makes us go deep into one branch before exploring others.

Time Complexity: O(V + E) where V = vertices, E = edges

Space Complexity: O(V) for visited set and recursion/stack

Use Cases: Cycle detection, pathfinding, topological sorting, maze solving

import java.util.*;

/**
 * Graph class with Depth-First Search (DFS) traversal
 * DFS explores as far as possible along each branch before backtracking
 */
public class GraphDFS extends Graph {
    
    /**
     * Depth-First Search - Recursive approach
     * Explores graph by going deep into branches first
     * @param start - starting vertex for traversal
     * @return List of vertices in DFS order
     */
    public List<String> dfsRecursive(String start) {
        List<String> result = new ArrayList<>();      // Store traversal order
        Set<String> visited = new HashSet<>();        // Track visited vertices
        dfsHelper(start, visited, result);
        return result;
    }
    
    /**
     * Helper method for recursive DFS
     * @param vertex - current vertex being processed
     * @param visited - set of already visited vertices
     * @param result - list accumulating the traversal order
     */
    private void dfsHelper(String vertex, Set<String> visited, List<String> result) {
        // Base case: null vertex or already visited
        if (vertex == null || visited.contains(vertex)) return;
        
        // Process current vertex
        visited.add(vertex);    // Mark as visited
        result.add(vertex);     // Add to result
        
        // Recursively visit all unvisited neighbors
        List<String> neighbors = getNeighbors(vertex);
        for (String neighbor : neighbors) {
            if (!visited.contains(neighbor)) {
                dfsHelper(neighbor, visited, result);
            }
        }
    }
    
    /**
     * Depth-First Search - Iterative approach using Stack
     * Uses explicit stack instead of recursion call stack
     * @param start - starting vertex for traversal
     * @return List of vertices in DFS order
     */
    public List<String> dfsIterative(String start) {
        List<String> result = new ArrayList<>();
        Set<String> visited = new HashSet<>();
        Stack<String> stack = new Stack<>();
        
        // Start with the initial vertex
        stack.push(start);
        visited.add(start);
        
        // Process vertices from stack (LIFO - Last In First Out)
        while (!stack.isEmpty()) {
            // Pop vertex from stack
            String current = stack.pop();
            result.add(current);
            
            // Push all unvisited neighbors onto stack
            List<String> neighbors = getNeighbors(current);
            for (String neighbor : neighbors) {
                if (!visited.contains(neighbor)) {
                    visited.add(neighbor);  // Mark as visited when adding to stack
                    stack.push(neighbor);
                }
            }
        }
        
        return result;
    }
    
    /**
     * Main method - demonstrates DFS traversal
     */
    public static void main(String[] args) {
        GraphDFS graph = new GraphDFS();
        
        // Build a sample graph
        //      A --- B --- D
        //      |     |     |
        //      C --- E --- F
        graph.addEdge("A", "B");
        graph.addEdge("A", "C");
        graph.addEdge("B", "D");
        graph.addEdge("C", "E");
        graph.addEdge("D", "E");
        graph.addEdge("D", "F");
        graph.addEdge("E", "F");
        
        // Test both DFS approaches
        System.out.println("DFS Recursive: " + graph.dfsRecursive("A"));
        // Example output: [A, B, D, E, C, F]
        
        System.out.println("DFS Iterative: " + graph.dfsIterative("A"));
        // Example output: [A, C, E, F, D, B] (may differ due to stack order)
        
        // Note: Order may vary depending on how neighbors are stored
        // Both are valid DFS traversals
    }
}

Graph Traversal: Breadth-First Search (BFS)

Algorithm Explanation:

Breadth-First Search (BFS): Explores all vertices at the current depth before moving to vertices at the next depth level.

BFS Algorithm (using Queue):

  1. Create an empty queue and add start vertex
  2. Mark start vertex as visited
  3. While queue is not empty:
    • Dequeue vertex from front (current)
    • Add current to result
    • For each unvisited neighbor of current:
      • Mark neighbor as visited
      • Enqueue neighbor

Shortest Path Algorithm (BFS-based for unweighted graphs):

  1. Create a queue with start vertex and a map to track previous vertex in path
  2. While queue is not empty:
    • Dequeue current vertex
    • If current equals destination:
      • Reconstruct path by following previous pointers backward
      • Reverse path and return it
    • For each unvisited neighbor:
      • Mark as visited
      • Set previous[neighbor] = current
      • Enqueue neighbor
  3. If destination never reached, return null (no path exists)

Key Idea: Queue (FIFO) ensures we explore all neighbors at current level before going deeper.

Time Complexity: O(V + E)

Space Complexity: O(V) for queue and visited set

Use Cases: Shortest path in unweighted graphs, level-order traversal, finding connected components

Why BFS finds shortest path: Because it explores vertices in order of increasing distance from start.

import java.util.*;

/**
 * Graph class with Breadth-First Search (BFS) traversal
 * BFS explores all vertices at current depth before moving to next depth
 */
public class GraphBFS extends Graph {
    
    /**
     * Breadth-First Search - Level by level traversal
     * Uses queue (FIFO) to process vertices
     * @param start - starting vertex for traversal
     * @return List of vertices in BFS order
     */
    public List<String> bfs(String start) {
        List<String> result = new ArrayList<>();       // Store traversal order
        Set<String> visited = new HashSet<>();         // Track visited vertices
        Queue<String> queue = new LinkedList<>();      // FIFO queue for processing
        
        // Start with initial vertex
        queue.offer(start);
        visited.add(start);
        
        // Process vertices level by level
        while (!queue.isEmpty()) {
            // Get next vertex from front of queue
            String current = queue.poll();
            result.add(current);
            
            // Add all unvisited neighbors to queue
            List<String> neighbors = getNeighbors(current);
            for (String neighbor : neighbors) {
                if (!visited.contains(neighbor)) {
                    visited.add(neighbor);    // Mark as visited
                    queue.offer(neighbor);     // Add to end of queue
                }
            }
        }
        
        return result;
    }
    
    /**
     * Find shortest path between two vertices using BFS
     * BFS guarantees shortest path in unweighted graphs
     * @param start - starting vertex
     * @param end - destination vertex
     * @return List representing path from start to end, or null if no path
     */
    public List<String> shortestPath(String start, String end) {
        // Map to track how we reached each vertex
        Map<String, String> previous = new HashMap<>();
        Set<String> visited = new HashSet<>();
        Queue<String> queue = new LinkedList<>();
        
        // Initialize with start vertex
        queue.offer(start);
        visited.add(start);
        previous.put(start, null);  // Start has no previous vertex
        
        // BFS to find path
        while (!queue.isEmpty()) {
            String current = queue.poll();
            
            // Check if we reached destination
            if (current.equals(end)) {
                // Reconstruct path by following previous pointers backward
                List<String> path = new ArrayList<>();
                String vertex = end;
                
                // Trace back from end to start
                while (vertex != null) {
                    path.add(0, vertex);        // Add to front of list
                    vertex = previous.get(vertex);
                }
                
                return path;
            }
            
            // Explore neighbors
            List<String> neighbors = getNeighbors(current);
            for (String neighbor : neighbors) {
                if (!visited.contains(neighbor)) {
                    visited.add(neighbor);
                    previous.put(neighbor, current);  // Remember how we got here
                    queue.offer(neighbor);
                }
            }
        }
        
        return null; // No path exists from start to end
    }
    
    /**
     * Main method - demonstrates BFS traversal and shortest path
     */
    public static void main(String[] args) {
        GraphBFS graph = new GraphBFS();
        
        // Build a sample graph
        //      A --- B --- D
        //      |     |     |
        //      C --- E --- F
        graph.addEdge("A", "B");
        graph.addEdge("A", "C");
        graph.addEdge("B", "D");
        graph.addEdge("C", "E");
        graph.addEdge("D", "E");
        graph.addEdge("D", "F");
        graph.addEdge("E", "F");
        
        // Test BFS traversal
        System.out.println("BFS: " + graph.bfs("A"));
        // Output: [A, B, C, D, E, F] - visits level by level
        
        // Test shortest path
        System.out.println("Shortest path A to F: " + graph.shortestPath("A", "F"));
        // Output: [A, B, D, F] or [A, C, E, F] - both are shortest paths
        
        // BFS guarantees this is a shortest path (minimum number of edges)
    }
}

Adjacency Matrix Representation

Algorithm Explanation:

Adjacency Matrix: A 2D array where matrix[i][j] represents the edge between vertex i and vertex j.

Structure:

  • Create a 2D array of size V × V (where V is max number of vertices)
  • matrix[i][j] = 0 means no edge between vertex i and j
  • matrix[i][j] = 1 means edge exists (for unweighted graph)
  • matrix[i][j] = weight means weighted edge with that weight
  • For undirected graphs: matrix[i][j] = matrix[j][i]

Adding a Vertex:

  1. Assign the next available index to this vertex
  2. Store mapping: vertex name ↔ index

Adding an Edge (Undirected):

  1. Get indices i and j for both vertices
  2. Set matrix[i][j] = weight
  3. Set matrix[j][i] = weight (for undirected)

Checking if Edge Exists:

  1. Get indices for both vertices
  2. Return true if matrix[i][j] ≠ 0

Finding Neighbors:

  1. Get index i for the vertex
  2. Loop through row i
  3. For each j where matrix[i][j] ≠ 0, add vertex j to neighbors

Pros: O(1) edge lookup, good for dense graphs

Cons: O(V²) space even for sparse graphs, O(V) to get all neighbors

import java.util.*;

/**
 * Graph represented using Adjacency Matrix
 * Uses 2D array where matrix[i][j] represents edge from vertex i to j
 */
public class GraphMatrix {
    private int[][] matrix;                      // 2D array for edges
    private Map<String, Integer> vertexMap;      // vertex name -> index
    private Map<Integer, String> indexMap;       // index -> vertex name
    private int numVertices;                     // maximum vertices
    private int nextIndex;                       // next available index
    
    /**
     * Constructor - Initialize matrix with max capacity
     * @param maxVertices - maximum number of vertices
     */
    public GraphMatrix(int maxVertices) {
        this.matrix = new int[maxVertices][maxVertices];
        this.vertexMap = new HashMap<>();
        this.indexMap = new HashMap<>();
        this.numVertices = maxVertices;
        this.nextIndex = 0;
    }
    
    /**
     * Add a vertex to the graph
     * Assigns next available index to this vertex
     * @param vertex - vertex name to add
     */
    public void addVertex(String vertex) {
        // Only add if vertex doesn't exist and we have space
        if (!vertexMap.containsKey(vertex) && nextIndex < numVertices) {
            vertexMap.put(vertex, nextIndex);      // name -> index
            indexMap.put(nextIndex, vertex);       // index -> name
            nextIndex++;
        }
    }
    
    /**
     * Add an unweighted edge (weight = 1)
     * @param vertex1 - first vertex
     * @param vertex2 - second vertex
     */
    public void addEdge(String vertex1, String vertex2) {
        addEdge(vertex1, vertex2, 1);
    }
    
    /**
     * Add a weighted edge between two vertices
     * For undirected graph, sets both matrix[i][j] and matrix[j][i]
     * @param vertex1 - first vertex
     * @param vertex2 - second vertex
     * @param weight - weight of the edge
     */
    public void addEdge(String vertex1, String vertex2, int weight) {
        // Ensure both vertices exist
        addVertex(vertex1);
        addVertex(vertex2);
        
        // Get their indices
        int i = vertexMap.get(vertex1);
        int j = vertexMap.get(vertex2);
        
        // Set edge weight in matrix (undirected, so set both directions)
        matrix[i][j] = weight;
        matrix[j][i] = weight;
    }
    
    /**
     * Check if edge exists between two vertices
     * @param vertex1 - first vertex
     * @param vertex2 - second vertex
     * @return true if edge exists (matrix value != 0)
     */
    public boolean hasEdge(String vertex1, String vertex2) {
        Integer i = vertexMap.get(vertex1);
        Integer j = vertexMap.get(vertex2);
        
        // Return false if either vertex doesn't exist
        if (i == null || j == null) return false;
        
        // Edge exists if matrix value is non-zero
        return matrix[i][j] != 0;
    }
    
    /**
     * Get all neighbors of a vertex
     * Scans the row in matrix for non-zero values
     * @param vertex - vertex to find neighbors for
     * @return List of neighboring vertices
     */
    public List<String> getNeighbors(String vertex) {
        List<String> neighbors = new ArrayList<>();
        Integer i = vertexMap.get(vertex);
        
        if (i == null) return neighbors;
        
        // Check entire row for non-zero entries
        for (int j = 0; j < nextIndex; j++) {
            if (matrix[i][j] != 0) {
                neighbors.add(indexMap.get(j));
            }
        }
        
        return neighbors;
    }
    
    /**
     * Display the adjacency matrix
     * Shows matrix with vertex labels
     */
    public void display() {
        System.out.println("\nAdjacency Matrix:");
        
        // Print column headers
        System.out.print("   ");
        for (int i = 0; i < nextIndex; i++) {
            System.out.print(indexMap.get(i) + " ");
        }
        System.out.println();
        
        // Print each row with row label
        for (int i = 0; i < nextIndex; i++) {
            System.out.print(indexMap.get(i) + "  ");
            for (int j = 0; j < nextIndex; j++) {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println();
        }
    }
    
    /**
     * Main method - demonstrates adjacency matrix operations
     */
    public static void main(String[] args) {
        GraphMatrix graph = new GraphMatrix(5);
        
        // Add edges to build graph
        graph.addEdge("A", "B");
        graph.addEdge("A", "C");
        graph.addEdge("B", "D");
        graph.addEdge("C", "D");
        
        // Display the matrix
        graph.display();
        /* Output:
           A B C D
        A  0 1 1 0
        B  1 0 0 1
        C  1 0 0 1
        D  0 1 1 0
        */
        
        // Test operations
        System.out.println("\nNeighbors of A: " + graph.getNeighbors("A"));  // [B, C]
        System.out.println("Has edge A-C: " + graph.hasEdge("A", "C"));      // true
        
        // Pros: O(1) edge lookup, good for dense graphs
        // Cons: O(V²) space, O(V) to find all neighbors
    }
}

Topological Sort (for DAG)

Algorithm Explanation:

Topological Sort: A linear ordering of vertices in a Directed Acyclic Graph (DAG) such that for every directed edge (u, v), vertex u comes before v in the ordering.

Applications: Course scheduling, build systems, task dependencies, compilation order

Method 1: DFS-based Topological Sort

  1. Create an empty stack to store result
  2. For each unvisited vertex in the graph:
    • Call DFS starting from that vertex
  3. In DFS for a vertex:
    • Mark vertex as visited
    • For each unvisited neighbor:
      • Recursively call DFS on neighbor
    • After visiting all neighbors, push current vertex onto stack
  4. Pop all vertices from stack to get topological order

Method 2: Kahn's Algorithm (BFS-based)

  1. Calculate in-degree (number of incoming edges) for all vertices
  2. Add all vertices with in-degree 0 to a queue
  3. While queue is not empty:
    • Dequeue a vertex and add it to result
    • For each neighbor of this vertex:
      • Decrease its in-degree by 1
      • If in-degree becomes 0, add it to queue
  4. Result list contains topological ordering

Key Insight (DFS): A vertex is added to result only after all its descendants are processed.

Key Insight (Kahn's): We can only process a vertex when all its dependencies are done.

Time Complexity: O(V + E) for both methods

Note: Only works on DAGs (Directed Acyclic Graphs). If graph has a cycle, topological sort is not possible.

import java.util.*;

/**
 * Topological Sort for Directed Acyclic Graphs (DAGs)
 * Produces linear ordering where all edges go from left to right
 */
public class TopologicalSort {
    private Map<String, List<String>> adjacencyList;
    
    /**
     * Constructor - Initialize empty directed graph
     */
    public TopologicalSort() {
        this.adjacencyList = new HashMap<>();
    }
    
    /**
     * Add a vertex to the graph
     * @param vertex - vertex to add
     */
    public void addVertex(String vertex) {
        adjacencyList.putIfAbsent(vertex, new ArrayList<>());
    }
    
    /**
     * Add a directed edge (dependency)
     * @param source - prerequisite/dependency
     * @param destination - dependent item
     */
    public void addEdge(String source, String destination) {
        addVertex(source);
        addVertex(destination);
        adjacencyList.get(source).add(destination);
    }
    
    /**
     * Topological Sort using DFS approach
     * Vertices are added to result in reverse finishing time
     * @return List of vertices in topological order
     */
    public List<String> topologicalSort() {
        Stack<String> stack = new Stack<>();    // Stack for reverse order
        Set<String> visited = new HashSet<>();  // Track visited vertices
        
        // Process each unvisited vertex (handles disconnected components)
        for (String vertex : adjacencyList.keySet()) {
            if (!visited.contains(vertex)) {
                topologicalSortDFS(vertex, visited, stack);
            }
        }
        
        // Pop all vertices from stack to get topological order
        List<String> result = new ArrayList<>();
        while (!stack.isEmpty()) {
            result.add(stack.pop());
        }
        
        return result;
    }
    
    /**
     * DFS helper for topological sort
     * Pushes vertex onto stack AFTER visiting all descendants
     * @param vertex - current vertex
     * @param visited - set of visited vertices
     * @param stack - stack to build reverse order
     */
    private void topologicalSortDFS(String vertex, Set<String> visited, 
                                    Stack<String> stack) {
        // Mark current vertex as visited
        visited.add(vertex);
        
        // Recursively visit all unvisited neighbors first
        List<String> neighbors = adjacencyList.get(vertex);
        if (neighbors != null) {
            for (String neighbor : neighbors) {
                if (!visited.contains(neighbor)) {
                    topologicalSortDFS(neighbor, visited, stack);
                }
            }
        }
        
        // After visiting all descendants, push this vertex onto stack
        // This ensures dependencies are processed before dependents
        stack.push(vertex);
    }
    
    /**
     * Kahn's Algorithm for Topological Sort (BFS-based)
     * Repeatedly removes vertices with no incoming edges
     * @return List of vertices in topological order
     */
    public List<String> topologicalSortKahn() {
        // Calculate in-degree (number of incoming edges) for each vertex
        Map<String, Integer> inDegree = new HashMap<>();
        
        // Initialize all in-degrees to 0
        for (String vertex : adjacencyList.keySet()) {
            inDegree.put(vertex, 0);
        }
        
        // Count incoming edges for each vertex
        for (String vertex : adjacencyList.keySet()) {
            for (String neighbor : adjacencyList.get(vertex)) {
                inDegree.put(neighbor, inDegree.get(neighbor) + 1);
            }
        }
        
        // Queue for vertices with no dependencies (in-degree = 0)
        Queue<String> queue = new LinkedList<>();
        for (String vertex : inDegree.keySet()) {
            if (inDegree.get(vertex) == 0) {
                queue.offer(vertex);  // Can be processed immediately
            }
        }
        
        List<String> result = new ArrayList<>();
        
        // Process vertices with no remaining dependencies
        while (!queue.isEmpty()) {
            String current = queue.poll();
            result.add(current);  // Add to topological order
            
            // "Remove" this vertex by decreasing in-degree of neighbors
            List<String> neighbors = adjacencyList.get(current);
            if (neighbors != null) {
                for (String neighbor : neighbors) {
                    // Decrease in-degree (one dependency resolved)
                    inDegree.put(neighbor, inDegree.get(neighbor) - 1);
                    
                    // If all dependencies resolved, can process this vertex
                    if (inDegree.get(neighbor) == 0) {
                        queue.offer(neighbor);
                    }
                }
            }
        }
        
        // Note: If result.size() < number of vertices, graph has a cycle
        return result;
    }
    
    /**
     * Main method - demonstrates topological sort
     */
    public static void main(String[] args) {
        TopologicalSort graph = new TopologicalSort();
        
        // Example: Course prerequisites
        // Must take "Intro to CS" before "Data Structures"
        // Must take "Data Structures" before "Algorithms"
        // Must take "Discrete Math" before "Algorithms"
        
        graph.addEdge("Intro to CS", "Data Structures");
        graph.addEdge("Data Structures", "Algorithms");
        graph.addEdge("Intro to CS", "Discrete Math");
        graph.addEdge("Discrete Math", "Algorithms");
        
        // Get valid course orderings
        System.out.println("Topological Sort (DFS): " + graph.topologicalSort());
        // Possible output: [Intro to CS, Discrete Math, Data Structures, Algorithms]
        
        System.out.println("Topological Sort (Kahn): " + graph.topologicalSortKahn());
        // Possible output: [Intro to CS, Data Structures, Discrete Math, Algorithms]
        
        // Both orderings are valid - both respect all dependencies
        // Note: Multiple valid orderings may exist
    }
}

  • The Graph class uses an adjacency list (HashMap with ArrayLists), which is memory efficient for sparse graphs
  • The GraphMatrix class uses a 2D array for the adjacency matrix, which is faster for dense graphs
  • BFS is useful for finding shortest paths in unweighted graphs
